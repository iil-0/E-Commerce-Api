# E-Commerce API

.NET 9 tabanlı E-Ticaret REST API.

## Temel Özellikler

- **RESTful Endpoints:** Ürün, Kategori, Sipariş ve Kullanıcı yönetimi.
- **Güvenlik:** JWT tabanlı kimlik doğrulama.
- **Mimari:** Katmanlı ve Minimal API yapısı.
- **Veri:** PostgreSQL veritabanı ve Entity Framework Core.
- **Dokümantasyon:** Swagger UI entegrasyonu.

## Hızlı Başlangıç

1. API'yi çalıştırın.
2. `http://localhost:5185/swagger` adresine gidin.
3. **Auth** endpoint'inden token alarak diğer işlemleri test edin.

**Test Hesabı:** `admin@ecommerce.com` / `admin123`
